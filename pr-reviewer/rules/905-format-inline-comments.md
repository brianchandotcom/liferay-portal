# 905: Format Inline Comments

Begin an inline comment with a capital letter, and leave a blank line before and after it.

**Rationale:** The capital letter and the surrounding blank lines set a comment apart as a deliberate note rather than a fragment crammed against the code, and they make comments read consistently wherever they appear.

A violation is an inline comment that begins with a lowercase letter or that is not separated from the surrounding code by a blank line before and after.
